# ConcreTrack

## Descripción

Aplicación para registrar datos de ensayos de fractura para concreto. Diseñado para las necesidades de la empresa AJV Ingenieros S.A. 

## Tecnologías

- Software
    - Python 3.10.x
    - Tkinter
    - SQLite
- Hardware
    - Raspberry Pi 4 B (mínimo 4gb de Ram)
    - Pantalla táctil
    - Lector de huellas R503
